description: "Start a new academic paper writing session"
argument-hint: "<instruction> [--refs <dir>] [--bib <file>] [--examples <a,b>]"

This is a shortcut for `/write paper`. Invoke `/write` with `paper` as the content type, passing through all other arguments.

Run the `/write` command with the following arguments prepended:

```
paper $ARGUMENTS
```
