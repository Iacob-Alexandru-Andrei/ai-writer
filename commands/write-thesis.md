description: "Start a new thesis chapter writing session"
argument-hint: "<instruction> [--refs <dir>] [--bib <file>] [--examples <a,b>]"
---

This is a shortcut for `/write thesis`. Invoke `/write` with `thesis` as the content type, passing through all other arguments.

Run the `/write` command with the following arguments prepended:

```
thesis $ARGUMENTS
```
